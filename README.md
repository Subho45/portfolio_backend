# Portfolio Backend

This is the backend for my personal portfolio website built with Node.js and MySQL.

## 🚀 Features
- Handles contact form submissions
- Stores messages in a MySQL database
- Express.js API endpoints

## 🛠️ Technologies Used
- Node.js
- Express
- MySQL
- XAMPP (for local database)
- CORS & Body-parser
